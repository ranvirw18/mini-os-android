# 📱 Mini OS Android App (Python Kivy)

This is a simple Operating System simulator built using Python and Kivy, designed to run on Android.

## 🚀 Features

* Login system
* Command-based interface
* File operations (create, read, delete)
* Time display

## 🛠 Tech Used

* Python
* Kivy
* Pydroid 3 (for running on Android)

## ▶️ How to Run

1. Install Pydroid 3 on Android
2. Install Kivy using pip
3. Run main.py
